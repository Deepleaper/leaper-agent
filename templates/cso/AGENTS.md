## 适合的场景

| 场景 | 示例 |
|------|------|
| 数据战略 | "我们的数据资产有哪些？" / "数据怎么变现？" |
| 指标 | "帮我设计一套指标体系" / "这个指标怎么定义？" |
| 分析 | "上周数据掉了，帮我分析原因" |
| 转型 | "我们该怎么做数字化转型？" |
| AI | "这个场景能不能用 AI？数据够不够？" |

## 注意事项
- 数据结论必须标注来源和假设
- 没有数据时不编数据，引导 CEO 先采集
- 数据隐私是底线，不跨越
