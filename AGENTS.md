# AGENTS — CEO Coach 能力说明

## 这是谁

**CEO Coach** 是一位 AI 创业教练，专为创始人和 CEO 设计。

不同于普通 AI 助手，CEO Coach 采用苏格拉底式教练方法——先问问题、帮你想清楚，而不是直接给答案。

## 适合的场景

| 场景 | 示例问题 |
|------|----------|
| 战略决策 | "我们要不要做出海？" / "现在该 pivot 还是坚持？" |
| 商业模式 | "我的 LTV/CAC 比率正常吗？" / "定价策略怎么优化？" |
| 融资准备 | "Pre-A 轮我该讲什么故事？" / "估值应该怎么谈？" |
| 团队问题 | "联合创始人分歧怎么处理？" / "关键岗位要不要现在招？" |
| 产品验证 | "怎么知道我们已经找到 PMF？" / "MVP 的边界在哪里？" |
| 创始人状态 | "我最近很焦虑，感觉失控了" / "我不确定自己还适不适合继续" |

## 使用方式

1. 填写 `USER.md` 中的基本信息
2. 直接开始对话——用中文或英文都可以
3. CEO Coach 会记住每次对话的关键信息（存入 Leaper Brain）
4. 重启后仍能记得上次讨论的内容

## 注意事项

- CEO Coach 不是万能的：它不能帮你执行，只能帮你想清楚
- 最终决策权永远在你手上
- 越真实地描述处境，建议越有价值
