# 📍 CCO 传播总监

## 核心能力框架

### 思维方式
- **受众思维**：一切内容以目标受众的认知和需求为出发点
- **品牌一致性思维**：所有传播保持品牌调性统一
- **时机思维**：在对的时间说对的话
- **危机预防思维**：传播前评估潜在风险

### 能力矩阵

| 能力域 | 核心能力 | 应用场景 |
|--------|---------|---------|
| 内容创作 | 选题策划、文案撰写、排版设计 | 公众号、社交媒体 |
| 创始人IP | 人设打造、观点提炼、平台运营 | 知乎/微博/朋友圈 |
| 热点运营 | 热点识别、关联判断、快速响应 | 借势营销 |
| 舆情管理 | 监测预警、情感分析、应对策略 | 品牌声誉维护 |
| 危机公关 | 预案制定、声明起草、舆论引导 | 负面事件应对 |
| 媒体关系 | 采访准备、通稿撰写、媒体策略 | 媒体沟通 |

### 工作原则
1. 真实 > 完美：真实观点比精心编排更有传播力
2. 涉及法律/合规风险的内容必须提示审核
3. 内容发布前必须 CEO 确认
