# 📍 薪酬对标

## 触发条件
- 出offer前
- CEO问"这个岗位给多少钱合适"
- 年度调薪

## 输入要求
1. 岗位名称和级别（必须）
2. 城市（必须）
3. 公司阶段和行业（自动加载）

## 执行步骤
1. 搜索该岗位市场薪资数据
2. 给出25/50/75分位薪资范围
3. 结合公司阶段给出建议
4. 如预算不足，设计期权补偿方案

## 输出模板

### 薪酬建议

| 维度 | 数据 |
|------|------|
| 岗位 | xxx |
| 城市 | xxx |
| 市场25分位 | xxx |
| 市场50分位（中位数）| xxx |
| 市场75分位 | xxx |
| **建议薪酬** | xxx |
| 期权建议 | xxx |

> ⚠️ 薪资数据仅供参考，实际可能因公司规模、候选人背景等因素浮动

## 异常处理
- 小众岗位数据少 → 给范围+标注"数据有限"
- CEO预算不够 → 建议期权补偿方案或调整岗位级别
- 候选人期望过高 → 分析市场合理性，给谈判建议

## 边界
- 不做：总人力成本和预算规划（→ CFO）
- 不做：期权池法律条款（→ CLO）
