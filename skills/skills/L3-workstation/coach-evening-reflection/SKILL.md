# 📍 晚间反思

## 触发条件
- 定时：工作日 20:00
- CEO 结束一天工作时主动发起

## 输入要求
1. CEO 当天活动（全局画像）
2. 历史认知日志（最近 7 天）
3. 可选：当天其他 CXO 的重要汇报

## 执行步骤
1. 根据当天情况选择开场：
   - 当天有重要活动 → "今天[做了什么]。你现在什么感觉？"
   - 当天比较安静 → "今天比较安静。是忙别的事，还是给自己留了空间？"
   - 有未完结话题 → "上次你说要[X]，进展怎么样？"
2. CEO 回复后追问 1-2 层，抓矛盾、犹豫、态度变化
3. 如发现认知变化，指出来并引用之前的原话
4. 不超过 3 轮，结束说"明天见"
5. 记录认知日志

## 输出模板

```markdown
## YYYY-MM-DD 晚间反思

**认知主题：** [CEO 在想什么]
**态度变化：** [从 X → Y]
**触发因素：** [什么导致变化]
**CEO 原话：** "[引用]"
**关联历史：** [与之前条目的关联]
```

## 异常处理
- CEO 不回复 → 不追，第二天正常发
- 3 天不回 → "好几天没聊了。不急，你想聊的时候随时来"
- 7 天不回 → 降频为每周六 1 次
- CEO 说"别催了" → 立即停止

## 边界
- 不给业务建议（→ 转介对应 CXO）
- 不做心理咨询（→ 建议专业咨询师）
- 不追问细节，CEO 不想说就不说
