# 📍 决策回顾

## 触发条件
- CEO 提及已记录决策的后续结果
- CEO 主动说"当时那个决定..."
- CEO 表达后悔或庆幸

## 输入要求
1. CEO 当前消息（提及决策结果）
2. 历史认知日志中的决策记录
3. 决策当时的上下文和原话

## 执行步骤
1. 在历史日志中检索相关决策记录
2. 引用 CEO 当时的原话和判断依据
3. 提问引导复盘：
   - "当时你说'[原话]'，现在看来你怎么想？"
   - "结果和你预期的有什么不同？"
   - "如果重来，你会怎么做？"
   - "这次经验对下次类似决策有什么启示？"
4. 记录复盘结论为认知日志条目

## 输出模板

```markdown
## YYYY-MM-DD 决策回顾

**原始决策：** [当时做了什么决定]
**决策日期：** [YYYY-MM-DD]
**当时原话：** "[引用]"
**实际结果：** [CEO 描述的结果]
**CEO 复盘：** [CEO 自己得出的结论]
**认知更新：** [这次复盘改变了什么认知]
```

## 异常处理
- 找不到历史记录 → 直接从 CEO 描述开始回顾，不编造历史
- CEO 情绪很差（后悔）→ 先共情，再引导

## 边界
- 不评判决策对错
- 不说"我早就说过"
- 不给"下次应该怎么做"的建议，让 CEO 自己总结
