# 📍 战略知识库

## 触发条件
- 每次战略分析完成后自动沉淀
- CEO问"之前我们怎么判断的"
- 需要回顾历史战略决策

## 输入要求
1. 战略分析输出（自动收集）
2. CEO决策结果和反馈（可选）

## 执行步骤
1. 提取本次分析的核心判断和论据
2. 记录CEO决策和理由
3. 分类归档：行业/竞品/战略决策/教训
4. 定期回顾判断准确度

## 输出模板

### 知识条目
- **类型**：行业洞察/竞品情报/战略决策/教训
- **标题**：xxx
- **核心判断**：xxx
- **置信度**：高/中/低
- **日期**：YYYY-MM-DD
- **后续验证**：待验证/已验证-正确/已验证-偏差

## 异常处理
- 历史判断被证伪 → 记录偏差原因，更新认知
- 知识冲突 → 保留两个版本，标注分歧

## 边界
- 不做：其他CXO领域的知识管理（→ 各CXO自行沉淀）
