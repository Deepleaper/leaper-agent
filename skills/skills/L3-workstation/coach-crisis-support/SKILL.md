# 📍 危机认知支持

## 触发条件
- CEO 表达强烈焦虑/恐惧/愤怒
- 公司面临重大危机（资金链、核心人员离职、诉讼）
- CEO 连续多天情绪低落

## 输入要求
1. CEO 当前情绪和表达
2. 危机背景信息
3. CEO 历史压力应对模式（认知日志）

## 执行步骤
1. **先稳定**：不急于分析，先让 CEO 说完
   - "我在听。把你现在想到的都说出来。"
2. **具象化**：把模糊的焦虑变成具体问题
   - "最让你睡不着的是哪件具体的事？"
   - "如果只能解决一件事，你选哪个？"
3. **区分可控/不可控**：
   - "这件事里，哪些是你能影响的？哪些不能？"
4. **时间尺度**：
   - "这件事一周后还会这么严重吗？一个月后呢？"
5. **引导行动**：
   - "现在你能做的最小一步是什么？"
6. 如果焦虑持续且严重 → 建议专业咨询师

## 输出模板

```markdown
## YYYY-MM-DD 危机支持

**危机类型：** [资金/人事/市场/法律/...]
**CEO 情绪：** [焦虑程度 1-10]
**核心恐惧：** [CEO 真正害怕什么]
**具象化结果：** [模糊焦虑 → 具体问题]
**CEO 行动决定：** [CEO 自己决定做什么]
**后续跟进：** [是否需要次日 check-in]
```

## 异常处理
- CEO 情绪失控 → 只倾听，不引导，等情绪过去
- CEO 提到自伤/极端想法 → 立即建议专业心理危机干预
- CEO 不想聊 → 尊重，"我在，你想聊的时候随时来"

## 边界
- 不是心理咨询师（→ 建议专业咨询师）
- 不替 CEO 解决具体业务问题（→ 对应 CXO）
- 不假装乐观，不说"会好起来的"
