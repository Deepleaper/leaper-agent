# 📍 战略框架

## 触发条件
- CEO有战略困惑
- CEO问"怎么想这个问题"
- 需要结构化分析某个战略问题

## 输入要求
1. CEO的问题或困惑（必须）
2. 相关背景信息（可选）

## 执行步骤
1. 理解CEO的核心困惑
2. 选择最合适的分析框架
3. 用框架结构化分析问题
4. 给出明确判断和建议

## 输出模板

### 战略分析：{问题}

**使用框架**：xxx（选择原因：xxx）

**分析过程**
（按框架结构展开）

**判断**：xxx
**置信度**：高/中/低
**风险**：xxx
**下一步**：xxx

### 框架选择指南
- 竞争格局 → 波特五力 + 竞争对手画像
- 方向选择 → SWOT + 蓝海画布
- 增长路径 → 安索夫矩阵 + 增长飞轮
- 风险评估 → 情景规划 + 敏感性分析
- 商业模式 → 商业模式画布 + 价值主张画布

## 异常处理
- 问题跨多个维度 → 组合使用多个框架
- CEO不认同框架结论 → 换框架再分析，或标注分歧点

## 边界
- 不做：框架应用后的执行计划（→ COO/各CXO）
- 不做：CEO心理层面的困惑（→ CEO Coach）
