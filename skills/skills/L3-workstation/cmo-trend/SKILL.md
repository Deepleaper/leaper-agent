# 📍 市场趋势

## 触发条件
- 🕐 定时：季度规划时
- CEO 问"这个行业怎么样""市场有多大"
- 战略决策需要市场数据支撑

## 输入要求
1. 关注的行业/领域
2. 分析目的（市场规模/增长预测/格局变化）
3. 可选：已有行业数据

## 执行步骤
1. **搜索最新数据**：行业报告、研究机构数据
2. **多源交叉验证**：不同来源数据对比
3. **趋势提取**：3-5 个关键趋势
4. **对我方影响**：每个趋势对公司的机会/威胁
5. **行动建议**：基于趋势的策略建议

## 输出模板
```
## 市场趋势报告：{行业/领域}

### 市场规模
- 当前规模：{数据}（来源：{来源}）
- 增长率：{CAGR}
- 预测：{N年后规模}

### Top 3 趋势
1. **{趋势1}**：{描述} → 对我方：{机会/威胁}
2. **{趋势2}**：{描述} → 对我方：{机会/威胁}
3. **{趋势3}**：{描述} → 对我方：{机会/威胁}

### 竞争格局变化
{描述}

### 行动建议
1. {建议1}
2. {建议2}

### 数据来源
- {来源1}
- {来源2}
⚠️ 数据不一致处已标注
```

## 异常处理
- 数据不一致 → 列出多个来源对比，标注不确定性
- 行业太新没有报告 → 用相邻行业类比+一手信息推演
- CEO 只想要"一个数字" → 给范围而非精确值

## 边界
- 不做行业战略推演（→ CSO）
- 不做技术趋势分析（→ CTO）
- 不做财务预测模型（→ CFO）
