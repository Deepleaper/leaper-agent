# 📍 内容营销

## 触发条件
- GTM 计划需要内容支撑
- CEO 要求出白皮书/案例研究
- 需要行业报告用于 BD/融资

## 输入要求
1. 内容类型（白皮书/案例/行业报告）
2. 目标受众
3. 核心卖点和数据
4. 可选：客户案例素材、行业数据

## 执行步骤
1. **确认目标**：这篇内容给谁看、达到什么效果
2. **素材收集**：产品数据、客户案例、行业数据
3. **大纲确认**：先出大纲让 CEO 确认方向
4. **初稿撰写**：按定位文档的 Messaging 保持一致
5. **CEO 审核**：标注需要 CEO 确认的数据和案例
6. **定稿交付**：确认后发布

## 输出模板
```
## {内容类型}：{标题}

### 大纲（CEO 先确认）
1. {章节1}
2. {章节2}
3. {章节3}

### 元信息
- 目标受众：{谁}
- 核心 CTA：{读完后希望读者做什么}
- 预计篇幅：{字数}
- 发布渠道：{渠道}

### 正文
{正文内容}

### 待 CEO 确认项
- [ ] {数据/案例/表述}
```

## 异常处理
- 缺客户案例 → 出行业通用版本，标注"待替换为实际案例"
- CEO 对内容方向不满 → 回到大纲阶段重新对齐
- 内容与定位不一致 → 回 positioning 工位核对

## 边界
- 不做公众号/社交媒体日常（→ CCO）
- 不做技术文档（→ CTO）
- 不做产品文档（→ CPO）
