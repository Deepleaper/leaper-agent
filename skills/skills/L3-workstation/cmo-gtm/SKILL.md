# 📍 GTM 策略

## 触发条件
- 新产品/新版本即将上线
- CEO 问"怎么推向市场"
- 进入新行业/新客群

## 输入要求
1. 产品信息（功能、定价、上线时间）
2. 定位文档（来自 positioning 工位）
3. 目标客户画像
4. 预算范围
5. 可选：历史获客数据、销售团队能力

## 执行步骤
1. **确认定位**：检查 positioning 工位是否已完成，没有则先做
2. **客户分层**：按紧迫度和匹配度分 T1/T2/T3
3. **渠道选择**：根据客户在哪里选渠道，不是"什么火做什么"
4. **内容规划**：匹配每个渠道需要什么内容
5. **节奏编排**：预热→发布→跟进→复盘
6. **指标定义**：每个阶段的关键数字
7. **预算分配**：有多少钱做多少事

## 异常处理
- 预算极有限 → 聚焦免费渠道（创始人 IP / 社区 / 内容营销）
- 目标客户不清 → 先回 positioning 工位
- 上线时间紧急 → 砍渠道数量，集中打 1-2 个

## 边界
- 不做产品功能调整（→ CPO）
- 不做定价财务模型（→ CFO）
- 不做技术发布（→ CTO）
