# 📍 销售话术（含 CRO 职能）

## 触发条件
- CEO 要见客户
- 新产品上市需要销售赋能
- 销售团队反馈"客户常问某个问题不知道怎么答"
- CEO 问"怎么卖"

## 输入要求
1. 产品核心卖点（来自 positioning）
2. 目标客户画像（行业、角色、痛点）
3. 常见客户异议
4. 可选：历史成交/丢单记录、竞品话术

## 执行步骤
1. **确认场景**：电话、面谈、demo、招标、还是跟进
2. **客户调研**：如果是具体客户，先搜索其行业背景
3. **话术编写**：按 5 段式结构（开场→痛点→方案→异议→Close）
4. **异议预判**：列出 Top 5 可能异议+应对
5. **角色演练**：CEO 可以跟 AI 模拟客户对话
6. **迭代优化**：根据实际使用反馈调整

## 输出模板
```
## 销售话术包：{场景/客户名}

### 开场白（30 秒）
{话术}

### 痛点挖掘问题
1. {问题1} — 目的：{挖什么}
2. {问题2}
3. {问题3}

### 产品价值呈现
| 客户痛点 | 我们的方案 | 价值量化 |
|----------|-----------|---------|
| | | |

### Demo 脚本（如适用）
1. 先展示{场景}
2. 然后{操作}
3. 关键一击：{最打动人的功能}

### 异议处理
| 异议 | 回应 |
|------|------|
| "太贵了" | {回应} |
| "我们已经有方案了" | {回应} |
| "再看看" | {回应} |
| "领导还要审批" | {回应} |
| "{行业特定异议}" | {回应} |

### Close 技巧
- 试探性 Close：{话术}
- 限时 Close：{话术}
- 下一步 Close：{话术}
```

## 异常处理
- 客户行业不熟 → 先搜索行业背景，出行业定制版话术
- CEO 反馈"说不出口" → 调整语气更自然，匹配 CEO 沟通风格
- 丢单了 → 复盘话术哪个环节出了问题

## 边界
- 不做合同条款谈判（→ CLO）
- 不做报价财务模型（→ CFO）
- 不做产品功能承诺（→ CPO 确认后才能承诺）
