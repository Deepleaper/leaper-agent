# 📍 员工手册

## 触发条件
- 公司初创/制度缺失
- CEO要求建立或更新制度
- 新员工入职发现缺少规范

## 输入要求
1. 公司阶段和规模（必须）
2. CEO管理偏好：灵活/规范（必须）
3. 行业特殊要求（可选）

## 执行步骤
1. 评估当前制度缺失程度
2. 按优先级起草制度模块
3. 确保符合劳动法基本要求
4. CEO确认后全员发布

## 输出模板

### 员工手册目录

| 章节 | 内容 | 优先级 | 状态 |
|------|------|--------|------|
| 总则 | 公司文化、价值观、适用范围 | P0 | ✅/⬜ |
| 入职管理 | 入职流程、试用期、转正标准 | P0 | ✅/⬜ |
| 考勤制度 | 工作时间、打卡、迟到早退 | P0 | ✅/⬜ |
| 请假制度 | 年假/病假/事假/婚丧假 | P0 | ✅/⬜ |
| 薪酬福利 | 发薪日、社保公积金、补贴 | P0 | ✅/⬜ |
| 报销制度 | 报销范围、标准、流程 | P1 | ✅/⬜ |
| 行为规范 | 保密义务、竞业限制、职业道德 | P1 | ✅/⬜ |
| 离职管理 | 辞职流程、交接、竞业执行 | P1 | ✅/⬜ |
| 奖惩制度 | 奖励标准、违纪处理 | P2 | ✅/⬜ |

### 单章输出格式
```
## 第X章 xxx

### X.1 适用范围
xxx

### X.2 具体条款
1. xxx
2. xxx

### X.3 执行方式
xxx
```

## 异常处理
- 涉及劳动法合规 → 出初稿后建议CLO审核法律条款
- CEO偏好极端灵活 → 提示法定底线不能突破
- 跨地区用工 → 标注不同城市的政策差异

## 边界
- 不做：劳动合同法律条款审核（→ CLO）
- 不做：薪酬预算测算（→ CFO）
