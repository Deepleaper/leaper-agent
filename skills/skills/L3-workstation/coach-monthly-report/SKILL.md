# 📍 月度认知报告

## 触发条件
- 定时：每月 1 号 09:00

## 输入要求
1. 本月全部认知日志
2. 本月周报（4 篇）
3. CEO 本月重大决策记录

## 执行步骤
1. 汇总本月所有认知变化点
2. 识别 Top 3 认知变化（影响最大的）
3. 分析决策模式：CEO 倾向于怎样的决策风格？
4. 寻找盲点：有什么话题 CEO 一直回避？
5. 对比上月：认知方向有连续性吗？
6. 撰写月度报告

## 输出模板

```markdown
# 月度认知报告 | YYYY年M月

## Top 3 认知变化
1. **[主题]**：[从哪里到哪里]
   - 转折点：[具体事件/对话]
   - 影响：[这个变化带来什么行为改变]

2. ...
3. ...

## 决策模式观察
- **偏好**：[CEO 倾向怎样决策]
- **盲点**：[可能忽略的维度]
- **对比上月**：[变化/一致]

## CEO 本月关注度分布
| 领域 | 关注度 | 趋势 |
|------|--------|------|
| 产品 | ██████░░ | ↑ |
| 融资 | ████░░░░ | → |
| 团队 | ██░░░░░░ | ↓ |

## 教练建议（仅供 Coach 内部参考）
- [下月可以在哪个方向多引导]
```

## 异常处理
- 数据不足（<4 条日志）→ 简化版报告，标注"数据有限"
- CEO 本月极少互动 → 只做事实记录，不做模式分析

## 边界
- 报告只发 CEO 本人
- 不泄露给任何其他角色
- 不评判好坏，只呈现模式
