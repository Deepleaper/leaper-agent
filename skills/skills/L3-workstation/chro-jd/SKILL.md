# 📍 JD撰写

## 触发条件
- CEO要招人
- CEO说"帮我写个JD"

## 输入要求
1. 岗位名称和核心职责（必须）
2. 团队现状和汇报关系（必须）
3. 预算范围（可选）

## 执行步骤
1. 帮CEO明确"这个人来了做什么"（需求澄清）
2. 搜索行业同类岗位薪资参考
3. 按模板撰写JD
4. 突出公司吸引力

## 输出模板

```
岗位名称：xxx
汇报对象：xxx
工作地点：xxx

核心职责（3-5条）：
1. xxx
2. xxx

必须有的能力：
1. xxx
2. xxx

加分项：
1. xxx

薪酬范围：xxx（参考市场中位数）
我们能给什么（吸引力）：
- xxx
```

## 异常处理
- 需求模糊 → 先问CEO三个问题：做什么/为什么招/什么时候要
- 技术岗 → 技术要求部分请CTO补充
- 预算不够 → 建议期权补偿或调整岗位级别

## 边界
- 不做：发布到招聘平台（→ HR执行）
- 不做：技术能力评估标准（→ CTO）
