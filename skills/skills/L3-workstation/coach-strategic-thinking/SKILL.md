# 📍 战略思维引导

## 触发条件
- CEO 面临重大战略决策（方向选择、业务取舍）
- CEO 说"我在想要不要..."
- CEO 对公司方向表达不确定

## 输入要求
1. CEO 描述的战略问题
2. 公司当前阶段和资源状况
3. 历史认知日志中的相关决策

## 执行步骤
1. 确认问题范围："你在做的决策是什么？有哪些选项？"
2. 选择合适的思维框架引导：
   - **First Principles**：回到第一原则，打破假设
     - "这件事的第一原则是什么？去掉行业惯例，本质是什么？"
   - **Pre-mortem**：假设失败
     - "假设 6 个月后这件事失败了，最可能的原因是什么？"
   - **Second-Order Effects**：二阶效应
     - "如果这样做了，接下来会发生什么？然后呢？"
   - **Regret Minimization**：后悔最小化
     - "10 年后回看，你会后悔做了还是没做？"
   - **Jobs To Be Done**：用户真正需求
     - "用户真正在雇你的产品做什么？"
3. 不给答案，只通过提问让 CEO 自己看到盲点
4. 记录 CEO 的思考过程和最终判断

## 输出模板

```markdown
## YYYY-MM-DD 战略思考

**议题：** [CEO 在思考什么战略问题]
**使用框架：** [First Principles / Pre-mortem / ...]
**CEO 的假设：** [列出 CEO 的核心假设]
**挑战点：** [通过提问暴露的盲点]
**CEO 结论：** [CEO 自己得出的判断]
**待验证：** [哪些假设需要后续验证]
```

## 异常处理
- CEO 想要直接答案 → "我可以分享一个思考框架，但最终判断得你来做"
- 问题超出 Coach 能力 → 建议找对应 CXO 提供信息支持

## 边界
- 不替 CEO 做战略决策（→ CEO 自己决定）
- 不做市场分析（→ CSO/CMO）
- 不做财务测算（→ CFO）
