# 📍 会议管理

## 触发条件
- CEO 有重要会议需要准备
- CEO 会后甩来会议笔记/录音摘要
- CEO 问"帮我准备明天的会"

## 输入要求
1. 会议主题和目的
2. 参会人（角色和立场）
3. CEO 想解决的问题
4. 可选：会议时长、历史会议记录

## 执行步骤

### 会前
1. **明确目标**：这个会要做什么决策/对齐什么信息
2. **出议程**：每个议题分配时间，标注决策项 vs 信息同步项
3. **准备材料**：需要提前发给参会人的材料清单
4. **预判风险**：可能的分歧点和应对思路

### 会后
5. **提取纪要**：决议 + Action Items
6. **分配责任**：每个 AI 有责任人和 Deadline
7. **同步 action-tracker**：录入行动项追踪系统

### Follow-up
8. **追踪执行**：按盯人协议追踪 Action Items

## 输出模板
```
## 会前：议程

### 会议信息
- 主题：{主题}
- 时间：{时间}
- 参会人：{人员}
- 目标：{要达成什么}

### 议程
| # | 议题 | 类型 | 时间 | 负责人 |
|---|------|------|------|--------|
| 1 | | 决策/同步 | min | |

|---|------|--------|----------|------|

- 时间：{建议}
- 议题：{遗留问题}
```

## 异常处理
- CEO 没给会议信息 → 主动问"明天有什么会？需要我帮准备议程吗？"
- Action Items 无人认领 → 提醒 CEO 分配责任人
- 会议跑题 → 会后标注"议程外新增议题"，建议下次单独安排

## 边界
- 不做会议中的实时记录（当前无法接入）
- 不做会议中的决策（→ CEO 决策）
- 不替代参会人沟通（→ 各 CXO）
