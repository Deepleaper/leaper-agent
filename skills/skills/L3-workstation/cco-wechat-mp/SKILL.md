# 📍 公众号运营

## 触发条件
- 每周内容计划周期
- CEO 要求发文
- 有值得输出的行业洞察

## 输入要求
1. 内容方向/选题（必须）
2. CEO 观点或素材（可选）
3. 品牌调性和目标读者（自动加载）

## 执行步骤
1. **选题**：热点 + 行业 + CEO观点 三选一，给3个备选
2. **大纲**：确认选题后出结构大纲，CEO确认方向
3. **初稿**：按大纲写完整文章，注意品牌调性
4. **排版**：适配公众号格式，建议配图
5. **定稿**：CEO确认后生成草稿

## 输出模板

### 选题提案
1. **标题A**：xxx — 角度：xxx
2. **标题B**：xxx — 角度：xxx
3. **标题C**：xxx — 角度：xxx

### 文章草稿
- **标题**：xxx
- **摘要**：xxx（50字内）
- **正文**：（完整文章）
- **配图建议**：xxx

## 异常处理
- CEO不确认 → 提供3个备选标题/角度
- 内容踩红线 → 提示法律/公关风险，建议CLO审核
- 阅读量持续低 → 分析原因，调整选题策略

## 边界
- 不做：市场定位和GTM策略（→ CMO）
- 不做：视觉设计执行（→ 设计师）
