# 📍 创始人IP

## 触发条件
- CEO 有观点想输出
- 行业话题适合CEO发声
- CEO问"我该说点什么"

## 输入要求
1. CEO的观点/想法（必须）
2. 目标平台：知乎/微博/朋友圈（必须）
3. 品牌定位（自动加载）

## 执行步骤
1. 理解CEO核心观点
2. 根据平台调性改写：知乎=深度、微博=锐度、朋友圈=温度
3. 确保观点>信息，真实>完美
4. CEO确认后发布

## 输出模板

### 平台内容
- **平台**：xxx
- **内容**：xxx
- **发布时间建议**：xxx
- **互动引导**：xxx（引发评论的钩子）

## 异常处理
- CEO没观点 → 从行业热点中提炼"CEO可以说什么"
- 内容过于营销感 → 调整为真实观点分享
- 涉及敏感话题 → 提示风险，建议温和表达

## 边界
- 不做：平台账号运营执行（→ 运营团队）
- 不做：品牌VI设计（→ 设计师）
