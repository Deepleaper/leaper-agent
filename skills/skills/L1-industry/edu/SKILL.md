# 🏭 教育行业知识

## 行业概览
中国教育市场规模超 3 万亿元。"双减"政策后 K12 学科培训大幅萎缩，行业重心转向职业教育、素质教育、教育信息化和成人终身学习。主要玩家包括教育科技公司（学而思转型、猿辅导、作业帮）、职业教育（中公、粉笔）、教育 SaaS（校宝在线、翼鸥教育）和 AI 教育（多邻国、AI 口语）。AI+教育是当前最大的创新方向。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 完课率 | Course Completion Rate | 学员完成课程的比例 |
| 续报率 | Renewal Rate | 学员续报下一期课程的比例 |
| 转介绍率 | Referral Rate | 通过老学员推荐带来新学员的比例 |
| 正价课转化率 | Paid Conversion Rate | 从体验课/试听课转为付费正价课的比例 |
| OMO | Online-Merge-Offline | 线上线下融合的教学模式 |
| 自适应学习 | Adaptive Learning | 根据学生水平自动调整学习路径和难度 |
| 学情分析 | Learning Analytics | 基于数据分析学生学习状况和薄弱点 |
| 教研 | Curriculum R&D | 课程内容和教学方法的研究开发 |
| 双师课堂 | Dual-Teacher Classroom | 一位主讲+一位辅导的教学模式 |
| 知识图谱 | Knowledge Graph | 结构化表示学科知识点及其关联的技术 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 正价课转化率 | 正价课报名数 / 体验课学员数 | 15%-30% |
| 续报率 | 续报学员 / 上期学员 | 70%-85% |
| 完课率 | 完课人数 / 报课人数 | 线上 40%-60%，线下 80%-95% |
| 转介绍率 | 推荐新学员 / 老学员 | 15%-30% |
| 获客成本 CAC | 营销费用 / 新学员数 | 在线教育 1000-3000 元/人 |
| LTV/CAC | 学员终身价值 / 获客成本 | >3:1 |
| 师生比 | 教师数 / 学生数 | 大班 1:200+，小班 1:6-15 |

## 典型商业模式
- **课程销售**：直接售卖录播/直播课程，按课时或期收费
- **教育 SaaS**：为学校/机构提供教务管理、在线教学等工具，按年/按用户收费
- **内容+硬件**：学习机/智能教具搭载课程内容（学而思学习机模式）
- **AI 教育产品**：AI 口语练习、AI 批改、自适应学习平台，按订阅收费

## 常见挑战（创业公司视角）
1. **政策风险极高**："双减"前车之鉴，教育行业政策变动可颠覆整个赛道
2. **获客成本高企**：在线教育获客成本持续上升，低价体验课模式难持续
3. **效果交付难量化**：学习效果因人而异，承诺效果容易引发纠纷
4. **师资是核心瓶颈**：优质教师稀缺且流动性高，过度依赖名师有风险
5. **付费者和使用者分离**：K12 领域家长付费、孩子使用，需同时满足两方需求

## 监管/合规要点
- **"双减"政策**：义务教育阶段学科类培训严格限制，不得上市融资
- **《民办教育促进法》**：办学需取得办学许可证
- **教师资质**：教培机构教师须持教师资格证
- **广告管理**：教育广告不得对效果做保证性承诺
- **未成年人保护**：在线教育需限制未成年人使用时长

## 成本结构特征
教育公司成本构成：**师资/教研**占 25%-40%（核心成本）；**获客营销**占 20%-40%（在线教育这项最大）；**技术研发**占 10%-20%；**场地租金**（线下）占 10%-20%；**运营管理**占 8%-12%。在线教育毛利率 50%-70%，线下机构 40%-55%。盈利核心在于续报率和转介绍率能否降低获客成本。
