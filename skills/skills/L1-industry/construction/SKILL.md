# 🏭 建筑行业知识

## 行业概览
中国建筑业总产值超 31 万亿元，是 GDP 的重要支柱。行业涵盖房建、基建、装饰装修、市政工程等细分领域。主要玩家包括央企建筑巨头（中建、中铁、中交）、地方国企、民营建企以及新兴建筑科技公司。行业正从劳动密集型向 BIM 数字化、装配式建筑、绿色建筑方向转型，但整体数字化程度仍处于较低水平。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| BIM | Building Information Modeling | 建筑信息模型，贯穿设计-施工-运维全生命周期的数字化技术 |
| 装配式建筑 | Prefabricated Construction | 工厂预制构件现场拼装，提升效率和质量 |
| EPC 总承包 | Engineering Procurement Construction | 设计-采购-施工一体化总承包模式 |
| 清单计价 | Bill of Quantities Pricing | 按工程量清单逐项计价的造价方式 |
| 工程变更 | Change Order | 施工过程中对设计或施工方案的修改 |
| 竣工验收 | Completion Acceptance | 工程完成后的质量和合规验收环节 |
| 资质等级 | Qualification Level | 建筑企业承揽工程的资质分级（特级/一级/二级/三级） |
| 垫资施工 | Advance-funded Construction | 施工方先垫付资金施工，业主后续付款 |
| 绿色建筑 | Green Building | 满足节能环保标准的可持续建筑 |
| 智慧工地 | Smart Construction Site | 利用 IoT、AI、5G 等实现工地数字化管理 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 产值利润率 | 净利润 / 建筑业总产值 | 3%-5%（行业偏低） |
| 项目毛利率 | 项目毛利 / 项目产值 | 8%-15% |
| 工期偏差率 | (实际工期 - 计划工期) / 计划工期 | ≤5% 为优秀 |
| 安全事故率 | 事故次数 / 百万工时 | <0.5 为优秀 |
| 应收账款周转天数 | 应收账款 / 日均收入 | 90-180 天（行业账期长） |
| 中标率 | 中标项目数 / 投标项目数 | 15%-30% |
| 工程质量合格率 | 质量验收合格项 / 总验收项 | 98%+ |

## 典型商业模式
- **施工总承包**：承揽建设项目的施工全过程，按工程量和合同结算
- **EPC 总承包**：提供设计+采购+施工一站式服务，利润率相对更高
- **建筑科技 SaaS**：提供 BIM、项目管理、造价管理等软件工具（广联达模式）
- **建材供应链平台**：连接建材供应商和施工方，赚取差价或服务费

## 常见挑战（创业公司视角）
1. **行业极度传统**：建筑企业数字化意愿低，决策者偏保守，推广周期极长
2. **现金流是命门**：垫资施工常态化，回款周期 6-12 个月甚至更长
3. **项目制收入不稳定**：大项目依赖严重，项目间隔期可能无收入
4. **关系驱动**：大型项目中标往往依赖行业关系和资质，纯技术优势难以突破
5. **人才短缺**：既懂建筑又懂技术的复合型人才稀缺

## 监管/合规要点
- **建筑业资质管理**：企业须取得住建部颁发的相应资质才能承揽工程
- **《建筑法》《建设工程质量管理条例》**：工程质量终身责任制
- **安全生产**：施工现场安全管理严格，事故追责到人
- **招投标法**：政府及国企项目必须公开招投标
- **环保要求**：施工扬尘、噪音、废料处理等需满足环保标准

## 成本结构特征
建筑施工企业成本构成：**人工成本**占 25%-35%（劳务分包为主）；**材料成本**占 40%-50%（最大项，含钢材、水泥、混凝土）；**机械设备**占 8%-12%；**管理费用**占 5%-8%；**财务成本**（垫资利息）占 2%-5%。净利润率通常仅 3%-5%，属于低利润高周转行业。
