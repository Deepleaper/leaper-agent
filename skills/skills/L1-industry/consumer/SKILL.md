# 🏭 消费品行业知识

## 行业概览
中国消费品市场社会消费品零售总额超 47 万亿元，涵盖食品饮料、美妆个护、服装鞋帽、家居日化等品类。主要玩家包括国际品牌（宝洁、联合利华、欧莱雅）、国货崛起品牌（元气森林、花西子、完美日记）、渠道商和 DTC 新品牌。消费升级与消费分级并存，新渠道（直播、社区团购）和新人群（Z 世代、银发经济）驱动品类创新。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 直面消费者 | DTC (Direct-to-Consumer) | 品牌直接触达消费者，跳过中间渠道 |
| 复购率 | Repurchase Rate | 已购用户再次购买的比例 |
| 渠道渗透率 | Channel Penetration | 产品在各类渠道的覆盖程度 |
| 货架占有率 | Shelf Share | 品牌在终端货架上占据的空间比例 |
| 品类 | Category | 按消费者需求划分的产品大类 |
| SKU | Stock Keeping Unit | 最小库存管理单元（单品） |
| 动销率 | Sell-through Rate | 已铺货 SKU 中实际产生销售的比例 |
| 经销层级 | Distribution Tiers | 从品牌到消费者之间的渠道层数 |
| 私域流量 | Private Domain Traffic | 品牌自有的可反复触达的用户池 |
| 新品成功率 | New Product Success Rate | 新推出产品达到销售预期的比例 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 毛利率 | (收入 - COGS) / 收入 | 快消品 40%-60%，美妆 60%-80% |
| 复购率 | 重复购买用户 / 总购买用户 | 30%-50%（高频品类可达 60%+） |
| 渠道费用率 | 渠道费用 / 渠道收入 | 线上 15%-25%，线下 20%-35% |
| 库存周转天数 | 平均库存 / 日均销售成本 | 食品 30-60 天，服装 60-120 天 |
| 新品贡献率 | 新品收入 / 总收入 | 20%-35% |
| 品牌知名度 | 调研认知率 | 目标市场 30%+ 为有效 |
| 营销 ROI | 营销带来的增量收入 / 营销支出 | 3:1 以上 |
| 客单价 | 总收入 / 订单数 | 同比持续增长 |

## 典型商业模式
- **品牌+渠道分销**：建立品牌，通过线上线下多渠道经销体系销售（传统模式）
- **DTC 直营**：自建官网/小程序/直播间直接销售，掌控用户数据和利润
- **代工/OEM + 自有品牌**：利用成熟供应链代工能力，快速推出自有品牌产品
- **订阅制**：周期性配送（鲜花、零食盒子），提高 LTV 和现金流可预测性

## 常见挑战（创业公司视角）
1. **品牌建设成本高**：从 0 到 1 建立品牌认知需大量营销投入，ROI 不确定
2. **渠道费用吞噬利润**：天猫/京东佣金+推广费可达 30%-40%，线下进场费同样高昂
3. **供应链管理**：库存过多滞销、过少断货，库存管理是永恒难题
4. **同质化竞争**：品类门槛低，爆品出来后跟风者快速涌入
5. **价格战压力**：平台大促频繁，消费者比价习惯养成，价格体系难维护

## 监管/合规要点
- **《食品安全法》**：食品类需取得生产许可证（SC 编号），标签规范严格
- **《化妆品监督管理条例》**：新原料和特殊功效化妆品需注册备案
- **《广告法》**：禁止"最"、"第一"等绝对化用语，功效宣传需有依据
- **产品质量法与三包规定**：退换货和质量保证要求
- **进口产品**：需检验检疫和海关备案

## 成本结构特征
典型消费品品牌成本构成：**原材料/生产成本**占售价的 20%-40%；**营销与品牌**占 20%-35%（新品牌前期可达 40%+）；**渠道费用**占 15%-25%；**物流仓储**占 5%-10%；**研发**占 2%-5%；**管理费用**占 5%-8%。净利润率：成熟品牌 10%-20%，新品牌前期普遍亏损。
