# 🏭 电商行业知识

## 行业概览
中国电商市场 GMV 超 15 万亿元，在线零售渗透率超 30%，全球领先。主要玩家包括综合平台（淘天、京东、拼多多）、兴趣电商（抖音、快手）、跨境电商（SHEIN、Temu、速卖通）和垂直电商（得物、小红书）。行业已进入存量竞争阶段，直播电商、内容电商、AI 电商和即时零售是新增长极。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 商品交易总额 | GMV (Gross Merchandise Volume) | 平台总成交金额（含退款） |
| 客单价 | AOV (Average Order Value) | 平均每笔订单金额 |
| 转化率 | Conversion Rate | 访客中完成购买的比例 |
| 获客成本 | CAC (Customer Acquisition Cost) | 获取一个新购买用户的成本 |
| 退货率 | Return Rate | 退货订单 / 总订单 |
| 付费流量 | Paid Traffic | 通过广告投放获得的访客流量 |
| 自然流量 | Organic Traffic | 搜索、推荐等免费渠道的访客流量 |
| 库存周转率 | Inventory Turnover | 一段时间内库存被售出和替换的次数 |
| 直播 GMV | Live Commerce GMV | 直播带货渠道的成交总额 |
| 跨境电商 | Cross-border E-commerce | 通过互联网实现跨国商品交易 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 转化率 | 成交订单 / UV | 品牌店 3%-8%，平台大盘 1%-3% |
| 客单价 AOV | GMV / 订单数 | 品类差异大，同比增长为健康 |
| 获客成本 CAC | 营销总费用 / 新客数 | 根据品类，50-200 元/新客 |
| 退货率 | 退货单数 / 总单数 | 服装 15%-30%，3C 3%-5% |
| 复购率 | 重复购买用户 / 总用户 | 30%-50% |
| 库存周转天数 | 平均库存 / 日均 COGS | 30-60 天（快消），60-120 天（服装） |
| 付费流量占比 | 付费 UV / 总 UV | <40% 为健康（过度依赖付费不可持续） |
| 净利润率 | 净利润 / 收入 | 自营 3%-8%，平台 15%-30% |

## 典型商业模式
- **平台模式（Marketplace）**：提供交易撮合，向商家收取佣金+广告费（淘宝/拼多多模式）
- **自营模式**：平台自行采购、仓储、销售，赚取进销差价（京东自营模式）
- **DTC 品牌电商**：品牌自建独立站/小程序直接销售，掌握用户资产
- **社交/内容电商**：通过内容种草+社交裂变驱动购买（小红书/抖音电商模式）

## 常见挑战（创业公司视角）
1. **流量成本持续攀升**：平台流量见顶，付费获客成本年均增长 15%-30%
2. **平台规则频繁变化**：算法调整、政策变动可能导致流量断崖式下降
3. **价格战和低价竞争**：拼多多效应下行业整体价格下行，利润空间被压缩
4. **供应链能力决定上限**：爆品时供不应求、滞销时库存积压，弹性供应链是核心能力
5. **退货率居高不下**：尤其服装品类，直播电商退货率可达 40%-60%

## 监管/合规要点
- **《电子商务法》**：经营者须办理市场主体登记，标明真实身份信息
- **《消费者权益保护法》**：七天无理由退货，虚假宣传处罚
- **《网络交易监督管理办法》**：直播带货需明确标注"广告"
- **跨境电商合规**：海关总署跨境电商进口正面清单，税收政策
- **数据与隐私**：用户数据收集、Cookie 使用需符合 PIPL

## 成本结构特征
电商品牌公司成本构成：**商品采购/生产成本**占 35%-50%；**平台佣金+推广费**占 20%-35%（最大可变成本）；**物流仓储**占 8%-15%；**人工**占 8%-12%；**退货损失**占 3%-8%。平台型企业则以技术研发和流量获取为主要成本，毛利率 60%-80%。
