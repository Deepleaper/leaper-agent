# 🏭 金融行业知识

## 行业概览
中国金融业增加值超 10 万亿元，涵盖银行、保险、证券、基金、支付、消费金融等细分领域。金融科技（FinTech）是数字化转型主线，主要玩家包括传统金融机构（工商银行、平安集团）、互联网金融平台（蚂蚁集团、京东科技）、垂直 FinTech（同盾科技、百融云创）。强监管是行业核心特征，持牌经营是底线要求。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 不良贷款率 | NPL Ratio (Non-Performing Loan) | 不良贷款余额占总贷款的比例 |
| 净息差 | NIM (Net Interest Margin) | 银行利息收入与利息支出的差额占生息资产的比例 |
| 风控模型 | Risk Model | 评估借款人/交易风险的量化模型 |
| KYC | Know Your Customer | 了解你的客户——合规身份验证流程 |
| AML | Anti-Money Laundering | 反洗钱合规体系 |
| 综合费率 | All-in Rate | 包含利息和各项费用的贷款实际年化成本 |
| 资产证券化 | ABS (Asset-Backed Securities) | 将贷款等资产打包为可交易的证券 |
| 联合贷款 | Co-lending | 金融科技公司与银行共同出资放贷 |
| 征信 | Credit Reporting | 个人或企业信用信息的记录和评估体系 |
| 支付牌照 | Payment License | 央行颁发的第三方支付业务许可证 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 不良率 | 不良贷款 / 总贷款 | 消费金融 1%-3%，小微贷 3%-6% |
| ROA | 净利润 / 总资产 | 银行 0.8%-1.2%，消金 2%-4% |
| ROE | 净利润 / 净资产 | 10%-18% |
| 获客成本 | 营销费用 / 新客户数 | 信用卡 100-300 元，消费贷 50-200 元 |
| 审批通过率 | 审批通过 / 申请总数 | 20%-40%（风险与规模的平衡） |
| 首逾率 (FPD30) | 首期逾期30天+ / 放款笔数 | <2% |
| 资金成本 | 融资综合利率 | 3%-6%（ABS/银行授信） |

## 典型商业模式
- **助贷/联合贷**：为银行等资金方提供获客和风控能力，按放款量收取服务费
- **支付服务**：提供聚合支付、跨境支付等，按交易笔数/流水抽取费率
- **金融 SaaS**：为金融机构提供风控系统、核心系统等技术产品，按订阅收费
- **保险科技**：线上保险分销或为保险公司提供核保/理赔科技服务

## 常见挑战（创业公司视角）
1. **牌照壁垒**：几乎所有金融业务需持牌经营，牌照获取难度大、成本高
2. **合规成本高**：监管要求频繁变化，合规团队和系统投入持续增长
3. **资金成本决定利润**：非银机构资金成本远高于银行，利润空间被挤压
4. **信用风险**：经济下行期不良率可能急剧上升，吞噬数年利润
5. **大客户依赖**：To B 金融科技收入常集中在少数大行/头部机构

## 监管/合规要点
- **持牌经营**：支付、小贷、消费金融、保险等均需对应牌照
- **《个人信息保护法》**：金融数据采集和使用需严格合规，征信需持牌
- **利率上限**：民间借贷利率不得超过 LPR 的 4 倍（约 14%-16%）
- **资本充足率**：金融机构需满足最低资本充足率要求
- **反洗钱（AML）/KYC**：客户身份识别和大额/可疑交易报告

## 成本结构特征
金融科技公司成本构成：**资金成本/风险准备金**占 30%-50%（信贷业务最大项）；**技术研发**占 15%-25%；**获客营销**占 15%-25%；**合规与法务**占 5%-10%；**运营与客服**占 8%-12%。纯 SaaS 模式毛利 60%-80%，助贷模式毛利 30%-50%（扣除资金成本和坏账）。
