# 🏭 广告技术（AdTech）行业知识

## 行业概览
广告技术行业是连接广告主、媒体和用户的数字化中间层，通过程序化购买、数据管理和效果优化实现精准广告投放。中国数字广告市场规模超 6000 亿元，主要玩家包括平台型（字节、腾讯、阿里妈妈）、独立 DSP/SSP（The Trade Desk、Xandr）、数据服务商（数美、极光）以及效果归因工具商。行业正从流量红利期转向精细化运营，AI 和隐私保护是两大驱动力。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 程序化购买 | Programmatic Buying | 通过自动化竞价系统实时购买广告位 |
| 需求方平台 | DSP (Demand-Side Platform) | 广告主侧的自动化购买平台 |
| 供给方平台 | SSP (Supply-Side Platform) | 媒体侧的广告位管理和售卖平台 |
| 实时竞价 | RTB (Real-Time Bidding) | 毫秒级的广告位实时拍卖机制 |
| 千次展示成本 | CPM (Cost Per Mille) | 广告展示 1000 次的价格 |
| 点击成本 | CPC (Cost Per Click) | 每次用户点击广告的价格 |
| 转化成本 | CPA (Cost Per Action) | 每次用户完成目标动作的价格 |
| 广告交易平台 | Ad Exchange | 连接 DSP 和 SSP 的交易市场 |
| 数据管理平台 | DMP (Data Management Platform) | 收集、整合、分析受众数据的平台 |
| 归因模型 | Attribution Model | 判定用户转化应归功于哪个广告触点 |
| 广告可见性 | Viewability | 广告实际被用户看到的比例 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 点击率 CTR | 点击数 / 展示数 | 信息流 1.5%-3%，搜索 3%-8% |
| 转化率 CVR | 转化数 / 点击数 | 电商 2%-5%，APP 下载 8%-15% |
| 广告投资回报率 ROAS | 广告带来的收入 / 广告花费 | 3:1 以上为健康 |
| 千次展示收入 eCPM | 总收入 / 展示数 × 1000 | 信息流 30-80 元，开屏 150-300 元 |
| 填充率 Fill Rate | 实际展示数 / 广告请求数 | 80%-95% |
| 广告可见率 | 可见展示 / 总展示 | 70% 以上 |
| 竞价胜率 Win Rate | 胜出竞价 / 参与竞价 | 15%-30% |
| 客户留存率 | 续费客户 / 总客户 | 月留存 85%+ |

## 典型商业模式
- **平台抽佣模式**：作为 DSP/SSP/Ad Exchange 抽取广告流水的 10%-30% 作为技术服务费
- **SaaS 订阅模式**：提供广告优化工具、创意管理平台，按月/年收取订阅费
- **效果分成模式**：按实际带来的转化效果（CPA/CPS）与广告主分成，风险共担

## 常见挑战（创业公司视角）
1. **流量获取成本高**：头部媒体流量被大平台垄断，独立 AdTech 公司难以获得优质流量
2. **隐私合规压力**：PIPL 实施后，用户数据采集和使用受限，传统 DMP 模式面临重构
3. **大客户依赖**：营收往往集中在少数大客户，单一客户流失可能导致收入断崖
4. **技术门槛与延迟要求**：RTB 竞价需要 50-100ms 内完成，对基础设施要求极高
5. **反欺诈挑战**：虚假流量和点击欺诈导致广告主信任度下降

## 监管/合规要点
- **《个人信息保护法》（PIPL）**：用户画像需明示同意，不得过度收集个人信息
- **《互联网广告管理办法》**：程序化广告必须标注"广告"字样，禁止弹窗强制推送
- **《数据安全法》**：数据出境需安全评估，涉及大量用户数据的广告平台需合规审查
- **行业自律**：中国广告协会互联网广告标准，MMA 中国移动广告标准

## 成本结构特征
典型 AdTech 公司成本构成：**流量采购成本**占收入的 50%-70%（最大项）；**技术基础设施**（服务器、带宽、CDN）占 10%-15%；**研发人员**占 15%-20%；**销售与运营**占 8%-12%。毛利率通常在 25%-45% 之间，纯平台型（不过流量）毛利可达 60%+。
