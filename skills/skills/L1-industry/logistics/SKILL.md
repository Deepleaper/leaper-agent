# 🏭 物流行业知识

## 行业概览
中国物流市场总规模超 350 万亿元（社会物流总额），物流费用占 GDP 约 14.5%。行业涵盖快递、快运、整车、冷链、仓储、跨境物流等。主要玩家包括快递巨头（顺丰、三通一达、极兔）、零担快运（安能、百世快运）、合同物流（京东物流、菜鸟）、科技物流（满帮、货拉拉、G7）。数字化调度、自动驾驶、无人仓和跨境物流是创新热点。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 时效 | Transit Time | 货物从发出到送达的时间 |
| 单票成本 | Cost per Parcel | 处理一票快递/货物的综合成本 |
| 妥投率 | Delivery Success Rate | 首次配送成功的比例 |
| TMS | Transportation Management System | 运输管理系统 |
| WMS | Warehouse Management System | 仓库管理系统 |
| 干线运输 | Trunk Line Transport | 城市间的长距离大批量运输 |
| 最后一公里 | Last Mile Delivery | 从末端网点到收件人的最终配送 |
| 装载率 | Load Factor | 车辆/集装箱的实际载货量 / 最大载重 |
| 冷链物流 | Cold Chain Logistics | 全程温控的运输和仓储体系 |
| 货运平台 | Freight Marketplace | 撮合货主和运力的数字化平台 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 单票成本 | 总运营成本 / 处理票数 | 快递 1.5-3 元/票，快运 0.3-0.6 元/kg |
| 妥投率 | 成功妥投 / 总配送 | 98%+ |
| 时效达成率 | 按时送达 / 总配送 | 95%+ |
| 装载率 | 实际装载 / 额定装载 | 干线 85%-95%，末端 60%-80% |
| 破损率 | 破损件数 / 总处理件数 | <0.1% |
| 单车日均收入 | 总运费 / 车辆数 / 天数 | 干线车 1500-3000 元/天 |
| 仓库周转率 | 年出库量 / 平均库存 | 因品类差异大，12-24 次/年为一般水平 |
| 人效 | 日处理件数 / 操作人数 | 分拣中心 2000-5000 件/人·天 |

## 典型商业模式
- **直营快递**：全链路自营（顺丰模式），服务品质高但资产重
- **加盟快递**：总部管干线+分拣，末端加盟商负责揽收派送（通达系模式）
- **货运撮合平台**：连接货主和司机，按交易抽佣或收取会员费（满帮模式）
- **合同物流/3PL**：为品牌客户提供定制化仓配一体解决方案，按单量/流水收费

## 常见挑战（创业公司视角）
1. **价格战惨烈**：快递单票价格持续下降，电商件已降至 1-2 元区间
2. **资产投入大**：车辆、仓库、分拣中心需要大量资金，轻资产切入点少
3. **规模效应壁垒**：物流网络效应极强，没有规模就没有成本优势
4. **人力密集**：末端配送高度依赖人力，管理数十万人的运营挑战巨大
5. **客户粘性低**：货主/电商商家对价格极度敏感，切换物流服务商成本低

## 监管/合规要点
- **快递业务经营许可**：从事快递需取得邮政管理部门颁发的许可证
- **道路运输许可**：从事货物运输需取得道路运输经营许可证
- **安全管理**：危化品运输需特殊资质，包裹安检制度
- **劳动关系**：快递员/司机的劳动权益保护是监管重点
- **跨境物流**：涉及海关报关、进出口许可、税收合规

## 成本结构特征
快递企业成本构成：**运输成本**（干线+末端配送）占 50%-60%；**人力成本**占 20%-30%；**场地与分拣中心**占 10%-15%；**包装与耗材**占 3%-5%。快递行业毛利率 10%-20%，净利率 3%-8%。货运平台毛利率较高（40%-60%），但需持续投入补贴获取运力和货源。
