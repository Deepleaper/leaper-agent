# 🏭 企业教练/咨询行业知识

## 行业概览
中国管理咨询与企业教练市场规模超 3000 亿元，年增速 10%-15%。行业包括战略咨询（McKinsey、BCG、本土的和君、北大纵横）、IT 咨询（埃森哲、IBM）、专业教练（ICF 认证教练）、企业培训（DDI、Franklin Covey）及新兴的数字化咨询。近年趋势：高管教练需求增长、AI 赋能咨询交付、从"卖报告"向"陪伴式落地"转型、组织发展（OD）和领导力发展成为热门细分。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 企业教练 | Executive/Business Coach | 通过对话引导帮助客户发现解决方案的专业角色 |
| 管理咨询 | Management Consulting | 为企业提供战略、运营、组织等专业建议 |
| 组织发展 | OD (Organization Development) | 系统性提升组织效能的理论与实践 |
| 人天单价 | Daily Rate / Man-day Rate | 咨询顾问每人每天的收费标准 |
| 可计费利用率 | Billable Utilization | 可计费工时 / 总可用工时 |
| 客户留存率 | Client Retention Rate | 续约/复购的客户比例 |
| 教练时数 | Coaching Hours | 教练与客户进行一对一对话的总时长 |
| 项目里程碑 | Project Milestone | 咨询项目中的关键交付节点 |
| 能力模型 | Competency Model | 定义岗位/角色所需能力的结构化框架 |
| ROI 评估 | ROI Assessment | 对咨询/教练投入产出比的量化衡量 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 可计费利用率 | 可计费工时 / 总工时 | 60%-75%（过高影响业务开发） |
| 人天单价 | 项目收入 / 投入人天 | 初级 3000-5000 元，高级 8000-20000 元 |
| 客户续约率 | 续约客户 / 到期客户 | >60% 为良好 |
| 项目毛利率 | (项目收入-直接成本) / 项目收入 | 40%-60% |
| 人均产值 | 总收入 / 顾问人数 | 80-200 万元/年 |
| 客户满意度 NPS | 推荐者% - 贬损者% | >50 为优秀 |
| 销售管线倍数 | 管线金额 / 季度目标 | >3x 为健康 |

## 典型商业模式
- **项目制咨询**：按项目范围报价，交付报告/方案，收取项目费（传统咨询模式）
- **人天/工时制**：按投入的顾问人天或教练小时计费
- **年度顾问/Retainer 模式**：按年收取固定顾问费，提供持续陪伴式服务
- **培训+认证模式**：开发标准化课程/认证体系，规模化交付（DDI、ICF 认证模式）
- **数字化产品+咨询**：SaaS 工具（测评、OKR、360 反馈）+ 配套咨询服务

## 常见挑战（创业公司视角）
1. **高度依赖人才**：核心顾问/教练离开可能带走客户，人才流失风险大
2. **规模化困难**：服务交付依赖专家时间，边际成本不趋零
3. **价值难量化**：客户难以直接衡量咨询/教练 ROI，续约决策偏主观
4. **销售周期长**：B2B 决策链长，从接触到签约通常 3-6 个月
5. **知识产权保护弱**：方法论和工具易被复制，壁垒靠品牌和信任

## 监管/合规要点
- **营业资质**：管理咨询无特殊牌照要求，但涉及财税、法律需对应资质
- **教练认证**：ICF（国际教练联盟）认证为行业金标准，国内无强制要求
- **数据保密**：客户商业信息、高管个人信息需严格保密，建议签署 NDA
- **劳动法合规**：顾问外包/兼职模式需注意劳动关系认定
- **广告法**：宣传"保证效果""行业第一"等绝对化用语违规

## 成本结构特征
咨询/教练公司成本构成：**人工成本**占 50%-65%（最大成本项，含顾问薪酬、奖金、社保）；**办公及差旅**占 10%-15%（项目驻场差旅频繁）；**营销及BD**占 8%-12%；**知识研发/课程开发**占 5%-10%；**管理及行政**占 5%-8%。行业净利润率 10%-20%，头部品牌可达 25%。
