# 🏭 人力资源行业知识

## 行业概览
中国人力资源服务市场规模超 2.5 万亿元，涵盖招聘、灵活用工、薪酬外包、培训、人力SaaS等细分领域。主要玩家包括综合人力集团（中智、外企德科、科锐国际）、招聘平台（BOSS 直聘、猎聘、智联）、HR SaaS（北森、Moka、飞书People）和灵活用工平台。灵活用工和 AI 招聘是增长最快的赛道。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 灵活用工 | Flexible Staffing | 非固定劳动关系的用工模式（外包、派遣、兼职） |
| 猎头 | Executive Search / Headhunting | 为企业寻找中高端人才的专业招聘服务 |
| 人事外包 | HRO (HR Outsourcing) | 将薪酬、社保、入离职等人事事务外包 |
| ATS | Applicant Tracking System | 招聘管理系统——管理简历筛选、面试流程 |
| 人才画像 | Talent Profile | 基于数据构建的理想候选人特征模型 |
| 人效 | HR Efficiency / Revenue per Employee | 每位员工创造的收入或产出 |
| 雇主品牌 | Employer Brand | 企业作为雇主在人才市场的品牌形象 |
| OKR/KPI | 目标管理/绩效考核 | 企业绩效管理的两种主流方法论 |
| 社保合规 | Social Insurance Compliance | 企业为员工依法缴纳五险一金的合规要求 |
| 人才盘点 | Talent Review | 系统评估组织内人才能力和发展潜力 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 招聘到岗周期 | 从需求提出到候选人入职的天数 | 普通岗位 20-30 天，高端岗位 45-90 天 |
| 简历转化率 | 面试邀约 / 投递简历数 | 8%-15% |
| Offer 接受率 | 接受 offer / 发出 offer | 70%-85% |
| 试用期通过率 | 转正人数 / 入职人数 | 85%-92% |
| 灵活用工毛利率 | (服务费 - 人力成本) / 服务费 | 6%-12% |
| 猎头费率 | 猎头费 / 候选人年薪 | 20%-30% |
| HR SaaS 续费率 | 续费客户 / 到期客户 | 75%-90% |
| 客单价 | 年收入 / 客户数 | SaaS 3-10 万/年，灵活用工按人头计 |

## 典型商业模式
- **灵活用工/劳务派遣**：为企业提供灵活劳动力，按人头或工时收取服务费（低毛利高流水）
- **猎头/RPO**：为企业招聘中高端人才，按成功入职人才年薪比例收费
- **HR SaaS**：提供招聘、绩效、薪酬、学习等模块，按员工数或企业规模订阅收费
- **薪税外包**：代发薪资、代缴社保公积金，按人头按月收取服务费

## 常见挑战（创业公司视角）
1. **灵活用工合规风险**：社保合规和劳动关系认定是核心法律风险
2. **HR SaaS 续费难**：中小企业人事管理需求简单，愿为 SaaS 付费的少
3. **业务季节性强**：春节后和秋季是招聘旺季，淡季业务量下降明显
4. **数据冷启动**：招聘平台需要同时有足够的岗位和简历，双边平台冷启动难
5. **AI 替代焦虑**：AI 筛选简历、AI 面试等技术可能替代传统猎头和招聘角色

## 监管/合规要点
- **《劳动合同法》**：劳务派遣需取得许可，且有"三性"（临时性、辅助性、替代性）限制
- **社保合规**：全面社保入税后，企业须依法为全部员工缴纳社保
- **《人力资源市场暂行条例》**：经营人力资源服务需行政许可
- **个人信息保护**：简历等个人信息的收集、存储、共享需合规
- **劳务派遣比例**：用工企业使用派遣员工不得超过用工总量的 10%

## 成本结构特征
灵活用工公司：**人力成本**占 85%-93%（员工薪酬+社保），毛利仅 6%-12%，薄利多销。猎头公司：**顾问薪酬+提成**占 50%-65%，毛利率 30%-50%。HR SaaS：**研发**占 35%-45%，**销售**占 25%-35%，毛利率 65%-80%。人力资源行业整体"大收入低利润"，灵活用工收入规模大但净利率仅 2%-5%。
