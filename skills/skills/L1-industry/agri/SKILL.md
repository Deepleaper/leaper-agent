# 🏭 农业行业知识

## 行业概览
中国农业总产值超 8 万亿元，正从传统小农经济向规模化、数字化、品牌化转型。主要玩家包括大型农企（新希望、北大荒）、农资电商（大丰收、农分期）、智慧农业服务商（极飞、丰疆智能）和农产品流通平台。土地流转加速、农村劳动力减少和食品安全需求是三大结构性驱动力。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 精准农业 | Precision Agriculture | 基于数据和技术实现精细化种植管理 |
| 土地流转 | Land Transfer | 农村土地经营权从个体农户转移到规模经营主体 |
| 农资 | Agricultural Inputs | 种子、化肥、农药、农机等农业生产资料 |
| 亩产 | Yield per Mu | 每亩土地的产出量，衡量种植效率 |
| 溯源系统 | Traceability System | 从田间到餐桌的全链路追踪体系 |
| 植保 | Crop Protection | 通过喷洒农药、生物防治等保护作物的技术 |
| 冷链物流 | Cold Chain Logistics | 农产品低温保鲜运输体系 |
| 订单农业 | Contract Farming | 先有订单再种植，降低滞销风险 |
| 农业物联网 | Agri-IoT | 传感器监控土壤、气象、作物生长状态 |
| 生鲜损耗率 | Fresh Produce Loss Rate | 农产品从采摘到终端消费的损失比例 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 亩均利润 | (亩产收入 - 亩均成本) | 粮食 200-500 元/亩，经济作物 2000-8000 元/亩 |
| 生鲜损耗率 | 损耗量 / 总产量 | 有冷链 5%-8%，无冷链 20%-30% |
| 农资复购率 | 复购农户 / 总服务农户 | 60%-80% |
| 土地利用率 | 实际种植面积 / 可用面积 | 85%-95% |
| 客单价 | 总营收 / 订单数 | 因品类差异大，同比增长 10%+ 为健康 |
| 单位面积投入产出比 | 产出价值 / 投入成本 | 2:1 - 4:1 |
| 交付及时率 | 按时交付订单 / 总订单 | 90%+ |

## 典型商业模式
- **农资 SaaS + 电商**：提供种植管理软件引流，通过农资销售变现（大丰收模式）
- **智慧农业解决方案**：无人机植保、IoT 监控等硬件+服务打包销售（极飞模式）
- **产地直供 / 订单农业**：连接规模种植户与 B 端采购商，赚取流通差价和服务费
- **农业金融**：基于种植数据为农户提供贷款和保险，赚取金融服务费

## 常见挑战（创业公司视角）
1. **获客成本高且分散**：农户地理分散、数字化程度低，线下地推成本高
2. **季节性现金流**：农业生产周期长，回款与收获季强绑定，现金流压力大
3. **标准化难**：农产品品质受天气、土壤等不可控因素影响，难以工业化标准化
4. **信任建立慢**：农户对新技术和新品牌接受度低，需要长期关系维护
5. **政策依赖强**：农业补贴和土地政策变动直接影响商业模式可行性

## 监管/合规要点
- **《农产品质量安全法》**：农产品生产需建立质量追溯制度
- **《农药管理条例》**：农药经营需取得许可证，使用需建立档案
- **《种子法》**：种子生产经营需取得许可，转基因种子管理严格
- **土地政策**：农村土地流转需遵守"三权分置"政策，不得改变土地用途
- **有机/绿色认证**：需通过农业农村部指定机构认证

## 成本结构特征
农业科技公司成本构成：**获客与地推**占 20%-30%（农村市场线下为主）；**农资/硬件采购成本**占 30%-50%（如涉及农资电商）；**技术研发**占 15%-20%；**物流仓储**占 10%-15%；**运营管理**占 8%-12%。纯 SaaS 模式毛利可达 60%+，涉及实物流通的毛利通常 15%-25%。
