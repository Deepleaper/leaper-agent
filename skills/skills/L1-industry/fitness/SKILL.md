# 🏭 健身行业知识

## 行业概览
中国健身市场规模约 7000 亿元（含器械、服务、补剂等），健身人口渗透率约 5%-7%（vs 美国 20%+），增长空间大。主要玩家包括传统健身房（一兆韦德、威尔仕）、新型健身品牌（乐刻、超级猩猩、Keep）、家庭健身（Peloton、Fiture）和健身内容平台。行业正从"预售卡模式"向"按次付费+内容驱动"转型，AI 健身教练和智能穿戴是新方向。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 月活会员 | MAU (Monthly Active Users) | 每月至少到店/使用一次的会员数 |
| 坪效 | Revenue per Sqm | 每平米面积产生的收入 |
| 私教课消率 | PT Session Consumption Rate | 已购私教课中实际上课的比例 |
| 预售模式 | Pre-sale Model | 提前销售年卡/多年卡获取现金流 |
| 团课 | Group Fitness Class | 多人同时参加的集体健身课程 |
| 会员留存率 | Member Retention Rate | 会员在到期后续费的比例 |
| 门店满载率 | Capacity Utilization | 高峰时段实际使用人数 / 最大容纳人数 |
| 次卡 | Pay-per-Visit Card | 按次付费的消费方式 |
| 沉睡会员 | Dormant Members | 付费但长期不到店的会员 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 月到店率 | 月到店会员 / 总有效会员 | 40%-60%（健康运营） |
| 坪效 | 月收入 / 门店面积(㎡) | 300-600 元/㎡·月 |
| 会员续费率 | 续费会员 / 到期会员 | 年卡 30%-50%，月卡 60%-75% |
| 私教转化率 | 购买私教会员 / 总会员 | 15%-30% |
| 私教课消率 | 已上课时 / 已购课时 | 70%-85% |
| 单店月营收 | 门店月总收入 | 新型小店 15-30 万，传统大店 50-150 万 |
| 人效 | 月营收 / 员工数 | 3-5 万元/人·月 |

## 典型商业模式
- **月付/次卡模式**：低门槛按月或按次收费，靠到店体验留存（乐刻模式）
- **年卡预售模式**：提前收取年费，利用现金流开新店（传统模式，风险高）
- **内容订阅**：线上健身课程/AI 教练按月订阅（Keep 模式）
- **私教工作室**：小面积、高单价、重服务的精品私教模式

## 常见挑战（创业公司视角）
1. **预售模式的现金流陷阱**：年卡预售现金流好看但负债率高，关店潮频发
2. **选址决定生死**：健身房是强地理位置依赖业务，选址失误几乎无法挽回
3. **教练流动性高**：优质教练稀缺且容易带走客户，人员管理是核心难题
4. **淡旺季明显**：1-3 月和 9-10 月是旺季，夏季反而是淡季，现金流波动大
5. **线上健身变现难**：用户为内容付费意愿低，线上转化变现率不高

## 监管/合规要点
- **预付费管理**：多地出台健身预付卡管理办法，限制预收款金额和退款规则
- **《体育法》**：高危险性体育项目经营需取得许可
- **消防安全**：健身场所消防验收要求严格
- **私教资质**：虽无强制要求，但行业认证（NSCA、ACE）渐成标准
- **消费者权益**：会员退款纠纷是监管投诉热点

## 成本结构特征
健身房成本构成：**房租**占 25%-40%（最大固定成本）；**人力成本**（教练+运营）占 25%-35%；**设备折旧**占 8%-12%；**水电物业**占 5%-8%；**营销获客**占 5%-10%。新型小店模型（200-500㎡）坪效高、投资回收期 12-18 个月；传统大店（2000㎡+）投资大、回收期 24-36 个月。
