# 🏭 制造业行业知识

## 行业概览
中国制造业增加值超 33 万亿元，占全球 30%，是全球制造业第一大国。涵盖电子制造、机械装备、汽车零部件、消费品制造等。主要玩家包括制造巨头（富士康、比亚迪、美的）、工业互联网平台（树根互联、卡奥斯）、智能制造方案商（汇川技术、新松机器人）。"智能制造"和"工业4.0"是转型方向，MES、数字孪生、工业 AI 质检是创业切入点。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| OEE | Overall Equipment Effectiveness | 设备综合效率——衡量设备利用率的核心指标 |
| MES | Manufacturing Execution System | 制造执行系统——连接 ERP 与车间的生产管理系统 |
| 数字孪生 | Digital Twin | 物理设备/产线的数字化镜像模型 |
| 良品率 | Yield Rate | 合格产品数 / 总生产数 |
| 精益生产 | Lean Manufacturing | 消除浪费、持续改善的生产管理体系 |
| 柔性制造 | Flexible Manufacturing | 快速切换产品型号的制造能力 |
| 供应链管理 | SCM (Supply Chain Management) | 原材料采购到成品交付的全链路管理 |
| 预测性维护 | Predictive Maintenance | 基于数据预测设备故障，提前维护 |
| 工业物联网 | IIoT (Industrial IoT) | 工厂设备互联互通的物联网体系 |
| 产能利用率 | Capacity Utilization | 实际产量 / 最大产能 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| OEE | 可用率 × 性能率 × 良品率 | 世界级 85%+，一般工厂 60%-75% |
| 良品率 | 合格品 / 总产出 | 电子制造 98%-99.5%，机械 95%-98% |
| 产能利用率 | 实际产量 / 设计产能 | 75%-90% |
| 库存周转率 | COGS / 平均库存 | 8-15 次/年 |
| 交付准时率 | 按时交付订单 / 总订单 | 95%+ |
| 单位制造成本 | 总制造成本 / 产出数量 | 同比下降 3%-5% 为健康 |
| 设备停机时间 | 非计划停机时间 / 总运行时间 | <5% |

## 典型商业模式
- **OEM/ODM 代工**：为品牌客户代工生产，按订单量和加工费收费
- **工业 SaaS/MES**：为工厂提供 MES、APS、质量管理等软件，按订阅收费
- **智能硬件+服务**：工业机器人、检测设备等硬件销售+运维服务
- **工业互联网平台**：连接上下游企业，提供供应链协同和数据服务

## 常见挑战（创业公司视角）
1. **工厂数字化水平参差不齐**：很多工厂连基础 IT 系统都缺乏，推广智能制造门槛高
2. **定制化需求重**：每个行业甚至每个工厂的流程不同，标准产品难以复用
3. **决策者不是用户**：老板决策采购，工人和车间主任使用，需求与体验脱节
4. **ROI 量化难**：制造业客户要求看到明确的成本节省或效率提升数据
5. **竞争者多样**：既有国际巨头（西门子、施耐德），又有本土集成商低价竞争

## 监管/合规要点
- **安全生产法**：工厂安全管理、特种设备使用需取得许可
- **环保排放标准**：废气、废水、噪音排放需达标，违规处罚严厉
- **产品质量标准**：ISO 9001 等质量管理体系认证是基本门槛
- **出口管制**：高端制造设备和技术可能涉及出口管制
- **能耗双控**：高耗能行业产能受限，需满足能效标准

## 成本结构特征
制造企业成本构成：**原材料**占 50%-70%（最大项）；**人工**占 10%-20%；**设备折旧**占 8%-15%；**能源**占 5%-10%；**管理费用**占 3%-5%。制造业净利润率普遍 5%-10%，高附加值制造（芯片、精密仪器）可达 15%-30%。工业 SaaS 公司毛利率 60%-75%，但获客和实施成本高。
