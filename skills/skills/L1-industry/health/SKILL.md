# 🏭 医疗健康行业知识

## 行业概览
中国医疗健康市场规模超 10 万亿元，涵盖医药、医疗器械、医疗服务、数字健康等细分领域。主要玩家包括药企（恒瑞、百济神州）、器械（迈瑞、联影）、互联网医疗（京东健康、阿里健康、微医）、数字健康（医渡科技、森亿智能）。集采政策重塑药品和器械价格体系，AI+医疗和创新药是创业热点。行业强监管、长周期、高壁垒。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 集采 | Volume-Based Procurement (VBP) | 国家/省级组织的药品和器械集中带量采购 |
| 一致性评价 | Bioequivalence Evaluation | 仿制药与原研药质量疗效一致性的评价标准 |
| 临床试验 | Clinical Trial | 新药/器械上市前的人体试验（I-IV 期） |
| DRG/DIP | Diagnosis Related Groups | 按疾病诊断相关分组的医保支付方式 |
| MAH | Marketing Authorization Holder | 药品上市许可持有人制度 |
| 处方外流 | Prescription Outflow | 医院处方流向院外药房的趋势 |
| 医疗 AI | Medical AI | AI 在影像诊断、病理分析、药物研发等场景的应用 |
| SaMD | Software as a Medical Device | 作为医疗器械的软件 |
| RWE | Real-World Evidence | 基于真实世界数据的临床证据 |
| CDSS | Clinical Decision Support System | 辅助医生诊疗决策的临床支持系统 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| 研发管线价值 | DCF 折现 / 管线估值 | 早期创新药公司核心估值指标 |
| 临床试验成功率 | 进入下一期 / 总启动项目 | I→II 期 60%-70%，II→III 期 30%-40% |
| 产品毛利率 | (收入 - COGS) / 收入 | 创新药 80%-90%，器械 50%-70%，仿制药 40%-60% |
| 院内覆盖率 | 已进院医院数 / 目标医院数 | 上市后 2 年达 30%-50% |
| 医保覆盖 | 产品是否进入医保目录 | 进医保是放量关键节点 |
| 获客成本（互联网医疗） | 营销费用 / 新注册用户 | 30-80 元/用户 |
| 复诊率 | 复诊用户 / 首诊用户 | 互联网医疗 20%-40% |

## 典型商业模式
- **创新药/器械研发销售**：高投入高回报，靠专利保护期内的独家销售获利
- **医疗 SaaS/数字化**：为医院提供 HIS、CDSS、AI 影像等系统，按项目或订阅收费
- **互联网医疗平台**：在线问诊+处方+药品配送闭环，靠药品销售和服务费变现
- **CRO/CDMO**：为药企提供研发外包（CRO）或生产外包（CDMO）服务

## 常见挑战（创业公司视角）
1. **合规门槛极高**：医疗器械注册证、药品临床审批耗时 2-5 年
2. **院内销售壁垒**：进入公立医院需经过漫长的招投标和医院准入流程
3. **集采压力**：产品一旦被纳入集采，价格可能下降 50%-90%
4. **付费方单一**：医保是主要付费方，医保控费直接影响企业收入
5. **AI 医疗商业化难**：临床验证周期长、医院付费意愿低、审批流程复杂

## 监管/合规要点
- **《药品管理法》**：药品全生命周期管理，MAH 制度
- **医疗器械分类管理**：I/II/III 类器械，III 类需 NMPA 注册审批
- **AI 医疗器械**：SaMD 纳入医疗器械监管，需取得注册证
- **数据合规**：医疗数据属于敏感个人信息，处理需特别保护
- **医保准入**：国家医保谈判年度进行，价格降幅通常 40%-60%

## 成本结构特征
创新药企成本构成：**研发**占收入的 20%-50%（早期可能无收入纯烧钱）；**生产成本**占 10%-20%；**销售推广**占 20%-40%（学术推广、医药代表）；**管理费用**占 8%-12%。医疗 SaaS 公司：研发 30%-40%，销售 25%-35%，实施交付 15%-20%。医疗行业整体回报周期长，但护城河深。
