# 🏭 媒体与内容行业知识

## 行业概览
中国数字内容和媒体市场规模超 3 万亿元，涵盖短视频、长视频、资讯、播客、数字阅读、游戏等。主要玩家包括短视频平台（抖音、快手）、长视频（爱优腾芒）、资讯平台（今日头条、微信公众号）、音频（喜马拉雅、小宇宙）和 MCN 机构。注意力经济是核心逻辑，AI 生成内容（AIGC）正深刻改变内容生产效率和形态。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| DAU/MAU | Daily/Monthly Active Users | 日活/月活用户数 |
| 用户时长 | Time Spent | 用户在平台上的平均使用时长 |
| 内容分发 | Content Distribution | 将内容推送给目标受众的机制（算法推荐/编辑推荐） |
| MCN | Multi-Channel Network | 签约并孵化多个创作者/KOL 的机构 |
| AIGC | AI-Generated Content | 利用 AI 生成文字、图片、视频等内容 |
| 完播率 | Completion Rate | 用户看完整条视频/文章的比例 |
| 互动率 | Engagement Rate | (点赞+评论+分享) / 曝光量 |
| 千次曝光收入 | RPM (Revenue Per Mille) | 每千次内容曝光带来的收入 |
| 付费率 | Paying User Ratio | 付费用户 / 总用户 |
| IP | Intellectual Property | 具有商业价值的内容知识产权 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| DAU | 日独立活跃用户数 | 垂直媒体 10 万+，大众平台 1000 万+ |
| 人均使用时长 | 总时长 / DAU | 短视频 90-120 分钟，资讯 30-60 分钟 |
| 次日留存率 | 次日回访用户 / 新增用户 | 40%-55% |
| 7 日留存率 | 7日回访 / 新增 | 20%-35% |
| 付费率 | 付费用户 / MAU | 长视频 10%-20%，知识付费 3%-8% |
| ARPU | 总收入 / MAU | 广告模式 5-15 元/月，订阅模式 15-30 元/月 |
| 内容生产成本 | 内容投入 / 内容产出量 | 差异大，AIGC 可降低 50%-80% |

## 典型商业模式
- **广告变现**：免费内容吸引流量，通过信息流广告、品牌广告变现（头条/抖音模式）
- **订阅付费**：优质内容/去广告体验按月订阅收费（爱奇艺/Netflix 模式）
- **内容电商**：内容种草+带货直播，赚取佣金和广告费（小红书/抖音电商）
- **IP 授权/衍生**：将优质 IP 授权给影视、游戏、衍生品开发

## 常见挑战（创业公司视角）
1. **用户注意力被头部平台垄断**：抖音一家占据大量用户时长，垂直平台获客极难
2. **内容生产成本高**：优质原创内容需要持续投入，且爆款不可预测
3. **创作者/KOL 流动性**：核心创作者出走可能带走粉丝和内容资产
4. **变现效率低**：中国用户为内容付费意愿远低于欧美，ARPU 值偏低
5. **内容审核压力**：信息内容安全要求严格，违规内容可能导致下架

## 监管/合规要点
- **《网络信息内容生态治理规定》**：平台需建立内容审核机制
- **《互联网新闻信息服务管理规定》**：新闻类内容需取得相应资质
- **网络视听许可证**：提供音视频服务需取得《信息网络传播视听节目许可证》
- **AIGC 管理**：AI 生成内容需标注，深度合成内容有专门管理规定
- **未成年人保护**：青少年模式、使用时长限制

## 成本结构特征
内容平台成本构成：**内容成本**占 30%-50%（版权采购+原创+创作者分成）；**带宽与基础设施**占 10%-20%；**研发**占 15%-25%；**营销获客**占 15%-25%；**内容审核**占 3%-5%。广告模式毛利率 40%-60%，订阅模式毛利率 30%-50%（内容成本高），MCN 机构毛利率 20%-35%。
