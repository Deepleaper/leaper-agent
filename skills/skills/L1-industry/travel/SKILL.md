# 🏭 旅游行业知识

## 行业概览
中国旅游市场年收入超 5 万亿元，国内旅游人次超 50 亿。后疫情时代旅游消费强势复苏，但消费结构发生显著变化：短途游、周边游、体验式旅游增长快，传统跟团游占比下降。主要玩家包括 OTA 平台（携程、美团、飞猪）、酒店集团（华住、锦江、首旅）、景区运营方、旅行社及新兴的内容种草平台（小红书、抖音）。出境游恢复、AI 行程规划、文旅融合是新热点。

## 核心术语
| 术语 | 英文 | 含义 |
|------|------|------|
| 在线旅游代理 | OTA (Online Travel Agency) | 提供在线预订机酒/门票的平台 |
| 每间可售房收入 | RevPAR (Revenue Per Available Room) | 酒店核心经营指标 = 平均房价 × 入住率 |
| 平均房价 | ADR (Average Daily Rate) | 已售客房的平均价格 |
| 入住率 | Occupancy Rate | 已售客房 / 可售客房 |
| 每位旅客收入 | Revenue Per Pax | 总收入 / 服务旅客人次 |
| 提前预订天数 | Booking Lead Time | 从预订到入住/出行的天数 |
| 佣金率 | Commission Rate | OTA 平台向供应商收取的分销佣金比例 |
| 文旅融合 | Culture-Tourism Integration | 文化资源与旅游产品深度结合 |
| 目的地营销 | Destination Marketing | 针对特定旅游目的地的推广活动 |
| 收益管理 | Revenue/Yield Management | 通过动态定价最大化收入 |

## 关键 KPI
| KPI | 计算方式 | 健康基准 |
|-----|---------|---------|
| RevPAR | ADR × 入住率 | 经济型 120-180 元，中高端 300-600 元 |
| 入住率 | 已售房晚 / 可售房晚 | 经济型 >80%，中高端 >65% |
| 获客成本 CAC | 营销费用 / 新客数 | OTA 平台 30-80 元/新客 |
| 复购率 | 回头客 / 总客户 | OTA 40%-60%，酒店会员 30%-50% |
| 取消率 | 取消订单 / 总订单 | 酒店 15%-25%，航班 5%-10% |
| 客单价 | 总 GMV / 订单数 | 国内游 500-2000 元，出境游 5000-15000 元 |
| 净推荐值 NPS | 推荐者% - 贬损者% | >40 为优秀 |

## 典型商业模式
- **OTA 平台模式**：撮合供需，向供应商收佣金（携程模式，佣金率 10%-25%）
- **酒店连锁/加盟模式**：品牌输出+管理标准化，收取加盟费和管理费（华住模式）
- **景区运营模式**：门票收入+二次消费（餐饮、纪念品、体验项目）
- **旅行社/定制游模式**：打包产品赚取差价或收取服务费
- **内容种草+交易闭环**：通过短视频/图文内容引流，直接转化预订（抖音旅行）

## 常见挑战（创业公司视角）
1. **季节性波动剧烈**：旺季资源紧张、淡季客源不足，现金流管理难
2. **OTA 平台依赖**：中小商家过度依赖平台流量，佣金侵蚀利润
3. **供应链碎片化**：机票、酒店、门票、地接等资源分散，整合难度大
4. **用户忠诚度低**：价格敏感型消费者多，品牌粘性弱
5. **突发事件影响大**：自然灾害、公共卫生事件可导致业务瞬间归零

## 监管/合规要点
- **《旅游法》**：规范旅行社经营、导游管理、旅游合同
- **旅行社经营许可证**：出境游业务需要额外资质和质保金
- **在线旅游经营服务管理规定**：OTA 平台信息公示、大数据杀熟禁令
- **景区等级评定**：A 级景区标准及年度复核制度
- **签证与出入境管理**：出境游涉及签证政策变动风险

## 成本结构特征
旅游企业成本构成因细分不同差异大。**OTA 平台**：技术研发占 15%-25%，营销获客占 30%-45%，人工占 15%-20%，毛利率 70%-85%。**酒店**：物业租金/折旧占 25%-35%，人工占 20%-30%，能耗及运营占 15%-20%，净利润率 8%-15%。**旅行社**：采购成本（机酒地接）占 70%-85%，毛利率仅 10%-20%。
